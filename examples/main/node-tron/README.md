## Node.js Example for integrating Rango-SDK for Tron Transactions

### Requirements:

- rango-sdk
- @rango-dev/signer-tron
- tronweb@beta
- Node.js >= 20

### How to run the code?

Set up your wallet in index.ts, then run following codes:

```sh
cd /path/to/example/
yarn
node --import=tsx index.ts
```
