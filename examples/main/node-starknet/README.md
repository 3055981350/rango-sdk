## Node.js Example for integrating Rango-SDK for Starknet Transactions

### Requirements:

- rango-sdk
- @rango-dev/signer-starknet
- starknet
- Node.js >= 20

### How to run the code?

Set up your wallet in index.ts, then run following codes:

```sh
cd /path/to/example/
yarn
node --import=tsx index.ts
```
